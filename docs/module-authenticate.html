Module: authenticate

Admin authentication module

This application supports two authentication strategies: local or via a JSON web token. The former is based on a username and password. Every time an admin user uses the local authentication strategy a new JSON web token is signed for this user.

Both authentication strategies are implemented via Passport.

(inner) createJwtStrategy() → {Object}

Create JSON web token strategy.

The JSON web token will be extracted from the authentication header as a bearer token. The strategy expects an environment variable JWTSECRET that holds a secret for encrypting and decrypting the JSON web token.
