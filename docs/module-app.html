Module: app

Application module

In this module, the Express application is initialized, third-party middleware is loaded, and routes are mounted.

If the NODE_ENV environment variable is set to development, cross-origin resource sharing (CORS) is enabled.

The routes provide the REST API for performing operations on the MongoDB that stores admin users, diary entries, and images; the object modeling is done via Mongoose.
